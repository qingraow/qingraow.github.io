数值数据

MATLAB的数值数据，是最基本的数据类型

  • 整型
  • 浮点型
  • 复数型

整型

  • 无符号整数
  • 带符号整数
  • 均有8位，16位，32位，64位之分
  • 共有八种整数类型

MATLAB提供了数据类型转换函数来实现数据类型的转换

  • uint8：将数值数据转换为无符号8位整数
  • int8：将数值数据转换为带符号8位整数

浮点型

  • 单精度型：4个字节
  • 双精度型：8个字节

转换

  • single：将其他类型的数据转换为单精度型
  • double：讲其他类型的数据转换为双精度型

复数型

复型数据包括实部和虚部两个部分，实部和虚部默认为双精度型，虚数单位可以用 i 或者 j 来表示

  • real：求复数的实部
  • imag：求复数的虚部

输出格式

format函数

format函数只影响数据的输出格式，不影响数据的计算和存储

  • format：回到默认格式

  • format long：15字长定点数，显示15位双精度，7位单精度（scaled fixed point）

  • format short 5字长定点数，显示5位(scaled fixed point format with 5 digits)
  • format short e 5字长浮点数
  • format long e 15字长浮点数
  • format hex 16进制
  • format bank 定点货币形式
  • format rat 小数分数表示
  • format + ＋，－，空格
  • format compact 压缩空格
  • format loose 包括空格和空行
  • format short g 5位定点或浮点格式。
  • format long g 对双精度，显示15位定点或浮点格式，对单精度，显示7位定点或浮点格式。

常用数学函数

  • 函数的自变量规定为矩阵变量，也可以是标量，标量本身就是矩阵变量的一种特例
  • 函数在运算时，是将函数逐项作用于矩阵的每个元素上，所以最后运算的结果是一个与自变量同型的矩阵

三角函数

  • 以弧度为单位的函数 sin cos tan
  • 以角度为单位的函数(加上d) sind cosd tand

abs

  • 求实数的绝对值
  • 求复数的模
  • 求字符的ASCII值

取整

  • fix：舍去小数取整
  • floor：向下取整
  • ceil：向上取整
  • round：四舍五入

isprime

判断是否为素数，是则返回1

变量

  • 变量本质上是内存单元的一个抽象、
  • Matlab中，变量名是以字母开头，后接字母，数字下划线，最多63个字符
  • 变量名区分字母的大小写
  • 标准函数名及命令必须用小写字母

预定义变量

matlab系统本身定义的变量

  • ans： 默认赋值变量
  • i 和 j：虚数单位
  • pi：圆周率
  • NaN：非数

保存

保存

  • 内存变量文件：用于保存Matlab工作区变量的文件，.mat

  • save：创建内存变量文件

  • load：装入内存变量文件

矩阵

建立

  • 直接输入，矩阵的元素用中括号括起来，同一行用逗号或分隔，不同行用分号分隔

  • 利用已经创建好的矩阵建立更大的矩阵

  • 利用实部矩阵和虚部矩阵构成复数矩阵

冒号表达式

linspace

结构矩阵和单元矩阵

结构矩阵

单元矩阵

元素的引用

  • 通过下标引用，如果引用超过原来矩阵的界限，将扩展原来的矩阵，没有赋值的地方置为0

  • 通过序号引用
    • MATLAB中，矩阵元素按列存储，首先存储第一列元素，然后再存储第二列元素……
    • 矩阵元素的序号就是矩阵矩阵元素在内存中的排序

  • 序号与下标是一一对应的，以M×N矩阵A为例，矩阵元素A(i, j)的序号为(j-1)×m+i
    • 矩阵元素的序号和下标可以通过sub2ind和ind2sub相互转换
  • sub2ind：行列转换为序号

  • ind2sub: 序号转换为行列

子矩阵

  • 利用冒号表达式获取子矩阵

    • A(i,:)：第i行的全部元素
    • A(:,j)：第j列的全部元素

  • end：表示某一维末尾元素下标

空矩阵删除矩阵中的元素

reshape

  • reshape(A,m,n)：在矩阵总元素保持不变的前提下，将矩阵A重新排列成m×n的二维矩阵
  • reshape函数只是改变原来矩阵的行数和列数，不改变原矩阵的元素个数及其存储顺序

其他

  • A(:)：将矩阵A的每一列元素堆叠起来，成为一个列向量

基本运算

matlab的基本运算都是对于矩阵而言的

加减运算

  • 若两矩阵同型，则运算时两矩阵的相应元素相加减
  • 若两矩阵不同型，则Matlab给出对应的错误信息
  • 标量也可以与矩阵进行加减运算，这时把标量和矩阵的每一个元素相加减

乘法运算

  • 矩阵A和B进行乘法运算，要求A的列数和B的行数相等，此时矩阵A和B是可乘的，或称A和B两矩阵的维数和大小相容
  • 如果两者的维数或大小不相容，则给出错误信息，提示用户两个矩阵是不可乘的

除法运算

  • 在Matlab中，有两种除法运算：右除/和左除\

  • 若A矩阵是非奇异方阵

    • B/A等效于B*inv(A)
    • A\B等效于inv(A)*B

  • 对于矩阵来说，右除和左除表示两种不同的除数矩阵和被除数矩阵的关系

乘方运算

  • 一个矩阵的乘方运算可以表示成A^x，要求A为方阵，x为标量

点运算

  • 两个元素进行点运算，是指他们的对应元素进行相关运算，要求两矩阵同型

关系运算

  • Matlab提供了六种关系运算符：< <= > >= == ~=
  • 当两个比较量是标量时，直接比较两数的大小，若关系成立，返回1
  • 当参与比较的量是两个同型矩阵的时候，每一个元素进行比较，最终返回一个同型只有0和1的矩阵
  • 当参与比较的是一个标量，而另一个是一个矩阵时，把标量和矩阵的每一个元素逐个比较

逻辑运算

  • & | ！

  • 算术运算、关系运算和逻辑运算中，算术运算的优先级最高，逻辑运算的优先级最低，

  • 逻辑非运算是单目运算，优先级比双目运算高

  • 参与运算的是两个同型矩阵，则按标量规则逐个运算

  • 如果一个是标量一个是矩阵，则矩阵的元素逐个与标量运算

字符串处理

字符串的建立

  • 在Matlab中，字符串是用单引号括起来的字符序列

  • 若字符串中的字符含有单引号，则该单引号需要用两个单引号来表示

  • 建立多行字符串，形成字符串矩阵

  • 例

字符串的执行

  • eval(s)：执行字符串

字符串的转换

  • abs 和 double 函数都可以用来获取字符串矩阵对应的ASCII数值矩阵
  • char 函数可以把 ASCII矩阵转换为字符串矩阵

字符串的比较

  • 关系运算符：要求长度相同，两个字符串的每个字符依次比较，返回一个01向量

  • 字符串函数
    • strcmp(s1,s2)：比较两个字符串
    • strncmp(s1,s2,n)：比较两个字符串的前n个字符
    • strcmpi(s1,s2)：忽略大小写比较
    • strncmpi(s1,s2,n)：忽略大小写， 比较两个字符串的前n个字符

查找和替换

  • findstr(s1,s2)：返回短字符串在长字符串中的开始位置
  • strrep(s1,s2,s3)：将字符串s1中的所有子字符串s2替换为s3

矩阵处理

特殊矩阵

  • 通用性的特殊矩阵
  • 用于专门学科的特殊矩阵

通用的特殊矩阵

  • zeros：产生0矩阵
  • ones： 产生1矩阵，即幺矩阵
  • eye：产生对角线为1的矩阵，当矩阵为方阵时，得到一个单位矩阵
  • rand：产生(0,1)均匀分布的随机矩阵
  • randn：产生均值为0，方差为1的标准正态分布随机矩阵

以上函数的调用格式：以zeros为例

  • zeros(m)：产生m×m零矩阵
  • zeros(m,n)：产生m×n零矩阵
  • zeros(size(A))：产生与矩阵A同样大小的零矩阵

用于专门学科的特殊矩阵

  • 魔方矩阵——Mafic Square
  • magic(n)

  • 范德蒙矩阵：vander(V) 生成以向量V为基础的范德蒙矩阵

  • 希尔伯特矩阵：hilb(n) 生成n阶希尔伯特矩阵

  • 伴随矩阵
    • compan(p) ： p是一个多项式的系数向量，高次幂系数排在前，低次幂系数排在后

  • 帕斯卡矩阵 ：pascal(n) 生成一个n阶帕斯卡矩阵

矩阵变换

对角阵

  • 对角矩阵：只有对角线上有元素的矩阵
  • 数量矩阵：对角线上元素相等的对角矩阵
  • 单位矩阵：对角线上元素都为1的对角矩阵

对角线的约定

提取矩阵的对角线元素

  • diag(A)：提取矩阵A主对角线元素，产生一个列向量
  • diag(A,k)：提取矩阵A的第k条对角线的元素，产生一个列向量

构造对角矩阵

  • diag(V)：以向量V为主对角线元素，产生对角矩阵
  • diag(V,k)：以向量V为第k条对角线元素，产生对角矩阵

三角阵

  • 上三角阵：矩阵的对角线以下的元素全为0
  • 下三角阵：矩阵的对角线以上的元素全为0

上三角矩阵

  • triu(A)：提取矩阵A的主对角线及以上的元素

  • triu(A,k)：提取矩阵A的第k条对角线及以上的元素

下三角矩阵

  • tril()：与上三角矩阵完全相同

转置

  • .'：转置运算
  • '：共轭转置，在转置的基础上取每个数的复共轭

旋转

  • rot90(A,k)：将矩阵A逆时针方向旋转90°的k倍，当k=1时可以省略

  • fliplr(A)：对矩阵A实施左右翻转
  • flipud(A)：对矩阵A实施上下翻转

求逆

  • inv(A)：求方阵A的逆矩阵

矩阵求值

行列式

  • det(A)：求方阵A所对应的行列式的值

秩

  • 矩阵的秩：矩阵线性无关的行数或列数
  • rank(A)：求矩阵A的秩

迹

  • 矩阵的迹：矩阵对角线元素之和，也等于矩阵的特征值之和
  • trace(A)：求矩阵A的迹

范数

  • 矩阵或向量的范数用来度量矩阵或向量在某种意义下的长度

向量的3种常用范数

  • 向量1—范数：向量元素的绝对值之和

    • norm(V,1)

  • 向量2—范数：向量元素平方和的平方根

    • norm(V)或norm(V,2)

  • 向量∞—范数：所有向量元素绝对值中的最大值

    • norm(V,inf)

矩阵的范数

  • 矩阵A的1—范数：矩阵列元素绝对值之和的最大值

  • 矩阵A的2—范数：A’A的最大特征值的平方根

  • 矩阵A的∞—范数：所有矩阵行元素绝对值之和的最大值

函数与向量的完全相同

条件数

  • 矩阵A的条件数：A的范数和A的逆矩阵的范数的乘积
  • 条件数越接近于1，矩阵的性能越好，反之，矩阵的性能越差

函数

  • cond(A,1)：计算A的1—范数下的条件数
  • cond(A)或cond(A,2)：计算A的2—范数下的条件数
  • cond(A,inf)：计算A的∞—范数下的条件数

特征值和特征向量

定义

  • 设A是n阶方阵，如果存在常数λ和n维非零列向量x,使得等式Ax=λx成立
  • 称λ为A的特征值
  • x是对应特征值λ的特征向量

函数

  • E=eig(A)：求矩阵A的全部特征值，构成向量E
  • [X,D]=eig(A)：求矩阵A的全部特征值，构成对角阵D，并产生矩阵X，X各列是相应的特征向量

稀疏矩阵

矩阵的存储方式

  • 完全存储方式
  • 稀疏存储方式：只存储矩阵的非零元素的值及位置（即行号和列号）
    • 利用稀疏存储方式，矩阵元素的存储顺序并没有改变，也是按列的顺序进行存储

创建和转换

  • A=sparse(S)：将矩阵S转化为稀疏存储方式的矩阵A
  • S=full(A)：将矩阵A转化为完全存储方式的矩阵S

  • sparse(m,n)：生成一个m×n的所有元素都是零的稀疏矩阵
  • sparse(u,v,S)：其中u、v、S是三个等长的向量
    • S是要建立的稀疏存储矩阵的非零元素
    • u(i)和v(i)分别是S(i)的行下标和列下标

  • B=spconvert(A)

存储方式

  • 无规则结构的稀疏矩阵
  • 有规则结构的稀疏矩阵
  • 带状稀疏矩阵：所有非零元素集中在对角线上的矩阵
    • [B,d]=spdiags(A)：从带状稀疏矩阵A中提取全部非零对角线元素，赋给矩阵B及其这些非零对角线的位置向量d
    • A=spdiags(B,d,m,n)：m,n为原带状稀疏矩阵的行数和列数，矩阵B的第i列即为原带状稀疏矩阵的第i条非零对角线，向量d为原带状稀疏矩阵所有非零对角线的位置

  • 单位矩阵的稀疏存储
    • speye(m,n) 返回一个m×n的稀疏单位矩阵

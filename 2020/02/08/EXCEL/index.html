公式运算符与编辑公式

  • Excel公式即为运算式，以等于号（=）开头。

  • 公式组成部分：数字，运算符，单元格引用、函数

求和函数SUM

  • 注意：文本数字应该转换为数字

SUM函数是一个数学和三角函数

  • 可将值相加。

  • 可以将单个值、单元格引用或是区域相加

  • 或者将三者的组合相加。

语法：SUM(number1,[number2],…) number1 （必需参数）要相加的第一个数字。

  • 该数字可以是 数字，单元格引用或 A3:A10 之类的单元格范围。

Excel常用函数

平均值

AVERAGE

  • 参数可以是数字，可以引用单元格，可以引用单元格区域，可以是公式函数计算的值。

  • 最多255个参数

例：AVERAGE（1,A1,A1:A18,B2+B3,SUM(C3:C4)）

AVERAGEA

  • 参数除了数字，单元格区域，还可以是文本或逻辑值
  • 逻辑值true代表1，false代表0,；文本代表0

条件平均值

AVERAGEIF

满足一定条件求平均值

参数：（单元格区域，条件(可以是数字，公式，文本，单元格)，实际计算平均的单元格区域）

AVERAGEIFS

  • 满足多条件求平均值
  • （实际计算平均的单元格区域，条件判断区域，条件(可以是数字，公式，文本，单元格)，条件判断区域，条件……）

逻辑函数

AND,OR,NOT

  • AND，判断多个条件成立
    例：AND(E2>=60,F2>=60)

  • OR，判断多个条件至少有一个成立
    例：OR(E2>=60,F2>=60)

  • NOT,对逻辑值求反
    例：NOT(E2>=60)

IF

根据条件判断，然后返回指定内容

  • 参数：(条件，如果为真时返回的值，如果为假时返回的值)

例：
IF（E2>=60,”及格”,”不及格”）

AND，判断多个条件成立
例：AND(E2>=60,F2>=60)

OR，判断多个条件至少有一个成立
例：OR(E2>=60,F2>=60)

条件求和

SUMIF

根据条件求和

SUMIF(条件判断区域，条件，实际求和区域)
例：SUMIF（B2:B4,”>=60”,C2:C4）

SUMIFS

多条件求和

SUMIFS(实际求和区域，条件判断的区域1，条件1，条件判断区域2，条件2……)
例：SUMIFS（D2:D4，B2:B4,”>=60”,C2:C4,”>=60”)

区域乘积求和

SUMPRODUCT

先元素相乘，后对积求和。

SUMPRODUCT(数组参数1，数组参数2，数组参数3…)

SUMPRODUCT（B2:B4,C2:C4,D2:D4）

  • 结果为 3*6 + 2*8

例：员工1的销售跑步机总额

TRUE和FALSE返回参数的逻辑值，TRUE代表的数值是1，FALSE代表的数值是0，可以数值运算的，如：

TRUE*1=1,FALSE*1=0。

舍入和截取

ROUND

对数值进行四舍五入

语法： ROUND(需要四舍五入的数值，保留几位小数)

例： ROUND（B2,2）,ROUND（B2,0）,ROUND（B2,-1）

  • 大于0，在小数点右边四舍五入
  • 等于0，四舍五入到整数
  • 小于0，小数点左边四舍五入。

INT

对数值进行向下舍入,保留整数

例：INT（B2）

TRUNC

数字截取，

TRUNC(数据，保留的小数位)

例：TRUNC（B2） TRUNC（B2,1） TRUNC（B2,2）

ROUNDUP

向上舍入,向前一位进1

ROUNDUP（数据，保留小数位）

例：ROUNDUP（B2，2）ROUNDUP（B2，0）ROUNDUP（B2，-1）

  • 大于0，在小数点右边向上舍入
  • 等于0，向上舍入到整数
  • 小于0，小数点左边向上舍入。

ROUNDDOWN

向下舍入,舍去不进1

ROUNDDOWN（数据，保留小数位）

例：ROUNDDOWN（B2，2）ROUNDDOWN（B2，0）ROUNDDOWN（B2，-1）

  • 大于0，在小数点右边向下舍入，等于0，向下舍入到整数，小于0，小数点左边向下舍入。

统计

COUNT

统计含有数字的单元格个数

语法：COUNT（数据参数1，数据参数2…..）

例：COUNT（B2:B4,C2:C4）COUNT（B2:C4）

COUNTA

统计非空单元格个数

语法：COUNTA（数据参数1，数据参数2……）

  • 最多30个数据参数。

例： COUNTA（B2:B4,C2:C4），COUNTA（B2:C4）

COUNTBLANK

统计空白单元格

COUNTBLANK（统计空白单元格区域）

COUNTIF

根据条件统计单元格个数

语法：COUNTIF（统计区域，条件）

例：COUNTIF（B2:C4，”871”）

COUNTIFS

统计满足多个条件的单元格个数

COUNTIFS（条件区域1，条件1,条件区域2，条件2……）

例：COUNTIFS（B2:B18，”>800”,B2:B18，”<900”）

最值

MAX

求最大值

MAX(参数1，参数2……)

例：MAX(D2:D29) MAX(IF(C2:C29=K1,E2:E29))

  • 输入完成后必须使用Ctrl+shift+enter组合键，根据条件求最大值

MIN

求最小值

MIN(参数1，参数2……)

例：MIN(D2:D29) MIN(IF(C2:C29=K1,E2:E29))

Ctrl+shift+ennter组合键，根据条件求最小值

行列

ROW

返回行号

ROW（单元格或区域）

例：ROW（）ROW（B2）

ROW（B2:B4）Ctrl+shift+enter组合键，使用数组公式。

  • 生成序号：”P-“&ROW（）-2
  • 生成重复序号：”P-“&INT((ROW(D1)-1)/3)+1

ROWS

返回行数

ROWS(单元格区域)

例：ROWS(B2:B9）

COLUMN

返回列标

COLUMN（单元格或区域）

例：COLUMN（） COLUMN（B2）

COLUMN（B:D）Ctrl+shift+enter组合键，使用数组公式。

COLUMNS

返回列数

COLUMNS（单元格区域）

例：COLUMNS（B:D） COLUMNS（B2:D3）

垂直查找

VLOOKUP

  • VLOOKUP中V表示方向垂直查找

  • 用途：在表格数组的首列查找值，然后返回当前所在行中其他值。

语法：VLOOKUP(1查找的值，2查找区域，3返回值所在列，4查找精度)

  • 第四个参数决定是精确查找还是模糊查找。
    • 0或FALSE表示精确查找
    • 1或TRUE表示模糊查找 （<= 返回最后一个符合条件的）
    • 第四部分省略时进行模糊查找。
  • 查找值可以是值也可以是单元格引用。

例：VLOOKUP(“P-1”，B2:C7，2，FALSE) 从B2到C7 查找值 “P-1”，并返回第二列对应的值

Excel数据处理，分析

相对引用和绝对引用

相对引用是Excel常用的一种引用方式。格式：列号行号
F4单元格上的公式为”=SUM(B4:E4)“，其中B2:E2中的数字”4“是随着行号的变化而自动变化得来的，这就是相对引用的特点。

绝对引用也是Excel常用的一种引用方式。其格式为：$列号$行号。加入$符号的，下拉或者右拉会固定

多表协同链接

链接公式结构

  • 同一工作簿：=工作名称!单元格地址
  • 不同同工作簿：=’工作簿存储地址[工作簿名称]工作名称’!单元格地址

引用其他工作簿的内容默认的是绝对引用。
